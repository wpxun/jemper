go context包   2019-03-08


整个 context 包原码就有一个文件即 context.go

  1. Context 接口

从这点可以看出，Context 库设计的目的：
(1) 退出通知机制
(2) 传递数据

  1. 四个重要的类型
  • emptyCtx 类型：无实际作用的 int 型，这是最原始的 Context 接口实现；
  • cancelCtx 类型：Context + 取消树（这里的“取消树”是一个完整的名词）；
  • timerCtx 类型：cancelCtx + 过期时间；
  • valueCtx 类型：Context + 键值对。

实例化上述四个类型的API函数：

  • Background()、TODO()：构造 emptyCtx；
  • WithCancel()：构造 cancelCtx；
  • WithDeadline()、WithTimeout()：一个是时间点，一个是时间段，时间段会转化为时间点；构造timerCtx；
  • WithValue()：构造valueCtx。
